# guide-portal-home

本仓库用于归档和发布多个独立 HTML 页面。所有页面均为静态文件，不依赖后端服务，可直接通过浏览器访问。

## 项目简介

这是一个通用型页面归档仓库，主要存放各类独立 HTML 文档。页面内容可能涵盖说明、指南、工具页面或其他静态展示用途。仓库不针对特定域名或网站，仅为方便页面管理和版本控制而建立。

## 目录结构

```
guide-portal-home/
├── docs/           # 文档或说明文件
├── pages/          # 归档的 HTML 页面
├── assets/         # 页面引用的静态资源（如图片、样式、脚本）
└── README.md       # 本文件
```

- **docs/**：存放与项目相关的说明文档。
- **pages/**：存放所有独立的 HTML 文件，按需组织子目录。
- **assets/**：页面所需的公共资源文件，建议按类型（img、css、js）分类存放。

## 页面归档说明

- 每个 HTML 页面应保持独立，可单独打开使用。
- 若页面引用外部资源，建议将资源存放至 `assets/` 目录下，避免依赖外部链接。
- 页面命名建议使用有意义的英文或拼音，避免使用特殊字符。
- 新增页面时，可在 `pages/` 下创建对应子目录，或直接放置文件。
